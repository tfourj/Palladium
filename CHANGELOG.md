# Palladium Changelog

## v1.3.0
- Add support for local bundle package installs
- Add ability to lock package versions
- Add contact information to About section
- Add alternate runtime patching options

## v1.2.2
- Fix PIP index deprecation warning
- Fix cookie file parsing and normalization
- Add cookie paste support
- Add support for curl_cffi
- Refactor package install workflow on fresh app install

## v1.2.1
- Added option to reorder and hide picker settings

## v1.2.0
- Added support for managing galleries
- Fixed background python interruption
- Fixed package update download race
- Added advanced settings toggle to manage WebKitJSI patching

## v1.1.1
- Added automatic package update toggle
- Moved package manager settings to a dedicated settings page.
- Added clipboard and local-file imports for URL allowlists.

## v1.1.0
- Added a Downloads tab for browsing saved downloads.
- Added saved media previews and playback.
- Added package branch source settings.
- Refactored Settings into grouped pages for User Interface, Download Settings, Storage, and About.

## v1.0.3
- Added allowlist feature to downloads

## v1.0.2
- Fixed background downloads
- Added notification permission check on app start
- Added package update check on app startup (disabled by default)
- Added download option settings (customize default download options)

## v1.0.1-b4
- Added support for downloading playlists.
- Replaced history link limit dropdown menu with an input field.
- Added website button to About section.
- Fixed partial log output.
- Fixed cancel action sheet.

## v1.0.1-b3
- Added support for importing cookies in Netscape format.
- Added support for future app localization.
- Added support for search, select, and copy in console UI.
- Added setting to customize how many links get saved in history view.
- Fixed FFmpeg progress streaming.
- Fixed partial console output.

## v1.0.1-b2
- Improved FFmpeg bridge and stability.
- Added progress reporting and better cancellation.
- Fixed HLS, thumbnails, and yt-dlp issues.
- Improved install check performance.
- Added general fixes and improvements.

## v1.0.1-b1
- Fixed ffprobe file path parsing when downloading videos with emojis or symbols in the title.
